# WOW Showcase Sites

Коллекция из **32 уникальных сайтов** с **расширенным контентом**:
- большой первый экран,
- рекламная зона,
- подробные секции (направления, преимущества, кейсы, тарифы, FAQ),
- нижняя бренд-зона «подбородок» с CTA.

## Быстрый запуск

```bash
python3 -m http.server 8080
```

Откройте любой сайт: `http://localhost:8080/showcase/<site-name>/`.

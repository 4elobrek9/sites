# WOW Showcase Sites

Коллекция из **32 уникальных мини-сайтов** на разные темы.

## Структура

- `showcase/<site-name>/index.html` — страница проекта.
- `showcase/<site-name>/assets/styles.css` — стили.
- `showcase/<site-name>/assets/app.js` — небольшая интерактивность.
- `templates/site-template` — эталонный шаблон структуры сайта.

## Быстрый запуск

```bash
python3 -m http.server 8080
```

После запуска откройте `http://localhost:8080/showcase/<site-name>/`.

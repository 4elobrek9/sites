# Папка шаблонов

Эта директория показывает, как должен выглядеть любой новый сайт в репозитории:

1. `site-template/index.html` — базовая разметка страницы.
2. `site-template/assets/styles.css` — единая структура CSS (переменные, layout, компоненты).
3. `site-template/assets/app.js` — минимальная интерактивность без зависимостей.

Копируйте `site-template`, переименовывайте папку и меняйте контент/стили под тему.

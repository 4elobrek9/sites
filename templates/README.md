# Папка шаблонов

Продвинутый `site-template` теперь включает:

1. **Header** с логотипом, навигацией и CTA-кнопкой.
2. **Ad zone** — отдельный блок под рекламу/партнеров/спецоффер.
3. **Main-секции**: hero, карточки оффера, FAQ.
4. **Chin (подбородок)** — нижняя бренд-зона с контактами, доверием и финальным CTA.

Использование: скопируйте `site-template` и замените тексты/цвета/сценарии.
